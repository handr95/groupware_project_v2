# groupware_project_v2 : SGW(Social Network Service & Team Group  Ward)


## 개발 취지

* 대학생 때 개발했던 프로젝트를 개인 공부용으로 재개발 하고자함.
* 현재 구동도 안되고 있는 프로젝트를 구동되도록 하며, 미구현되어 있는 기능들을 구현 하는것을 목표로 개발 시작~.~

## 기반 프로젝트

    * 업로드
    * 삭제
    * 목록 조회

    * 조회
